# 2006年3月23日 星期四 晴

今天晴天。

早上起来，心情还可以。

到教室的时候，陈大壮说明天周五了。

我说是啊。

他说时间过得真快。

我说是啊。

他说期中考试快到了。

我说我知道。

他说你能行吗？

我说不知道，但我要试试。

他说我也要努力了。

我说一起努力吧。

他说好。

上课的时候，老师讲新课。

我听着，觉得好像能听懂一点。

但有些地方还是不太懂。

我想找林小雅问，但她今天好像很忙。

下课的时候，她被人叫走了。

我没机会和她说话。

中午吃饭的时候，陈大壮说林小雅今天好忙。

我说是啊。

他说有人在找她问题目。

我说她成绩好，别人找她正常。

他说你也是。

我说我成绩不好。

他说你成绩不好不代表别人不能找你。

我笑了，谁找我？

他说我找你啊。

我说找我干嘛？

他说教你做题。

我说你成绩比我还差。

他说但我在进步啊。

我说那我们一起进步吧。

他说好。

下午放学的时候，林小雅走过来。

说江小鱼，今天又没机会和你说话。

我说没事，你挺忙的。

她说不好意思。

我说没事，你忙你的。

她说有问题可以来找我。

我说我知道。

她笑了，那你继续加油。

我说你也是。

回家的路上，我想今天。

没什么特别的，就是普通的周四。

林小雅还是很忙，我没机会和她说话。

但没关系，明天还有机会。

窗外面阳光很好，天空很蓝。

春天真的要来了。

三月二十三号，晴天，没什么特别的。

但明天就是周五了，周末快到了。

继续加油吧江小鱼。
