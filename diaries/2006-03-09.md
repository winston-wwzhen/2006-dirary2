# 2006年3月9日 星期四 多云

明天就是家长会了。

早上起来，我心里有点紧张。

不知道明天会发生什么。

到教室的时候，大家都在聊明天的事。

陈大壮说明天我爸妈都会来。

我说两个人都来？

他说是啊，我妈也要来。

我说那你惨了。

他说我也觉得。

上课的时候，老师还在讲新课。

但我听不太进去。

脑子里全是明天的事。

妈会来，她会听到我的成绩。

她会和老师聊什么？

老师会说什么？

我不知道。

下课的时候，林小雅走过来。

说江小鱼，你还好吗？

我说还好。

她说你看上去有点紧张。

我说明天就是家长会了。

她说没事的，没什么好怕的。

我说我知道。

她看着我，说你尽力了，这就够了。

我看着她，想她说得对。

但心里还是有点紧张。

中午吃饭的时候，陈大壮说我明天肯定要被骂。

我说你爸妈都来？

他说是啊。

我说那你真惨。

他说我能有什么办法。

我说尽力就好。

他说尽力有个屁用。

我说那你别想了。

他说能不想吗？

我继续吃饭，想陈大壮说得对。

尽力有什么用，成绩不好就是不好。

但林小雅说尽力就够了。

我想她说得对。

下午放学的时候，林小雅走过来。

说江小鱼，明天别紧张。

我说我会尽量。

她说你妈来了之后，让她看到你的努力。

我说我会的。

她说我相信你能做到。

我看着她，想她这么相信我，我不能让她失望。

回家的路上，我想明天。

家长会，妈会来。

我会让她知道，我在努力。

虽然成绩不好，但我在进步。

这就够了。

窗外面天多云，风有点冷。

我拉紧外套，继续走。

三月九号，周四，家长会前一天。

紧张，但也有点期待。

期待妈能看到我的努力。

期待明天能顺利。

加油吧江小鱼。
