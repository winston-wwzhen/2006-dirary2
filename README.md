# 2006年日记

一部虚构的2006年16岁青少年日记作品。

## 项目简介

本项目以2006年为时代背景，通过日记形式记录了一个16岁高中生——江小鱼的生活点滴。作品力求还原那个年代的语言风格、文化氛围和青春期特有的迷茫与躁动。

## 主要人物

- **江小鱼**：主角，16岁，清溪县第一中学高一（3）班学生，内向敏感的青春期少年
- **林小雅**：江小鱼的暗恋对象，同班同学，文静温柔
- **陈大壮**：江小鱼唯一算是朋友的人，讲义气的差生
- **老王**：班主任，语文老师

## 时代背景

作品融入了2006年的时代元素：

- **音乐**：周杰伦《十一月的萧邦》、孙燕姿、林俊杰
- **网络**：QQ聊天、QQ空间、新浪论坛、天涯社区
- **游戏**：CS、魔兽争霸、传奇、梦幻西游
- **影视**：《武林外传》、超级女声
