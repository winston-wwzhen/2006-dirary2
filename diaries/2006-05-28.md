# 2006年5月28日 星期日 晴

周日。

早上起来，阳光很好。

昨天去图书馆了，今天在家复习。

妈说今天好好复习，明天要上学了。

我说我知道。

我坐在书桌前，翻开数学书。

做题，一道一道地做。

林小雅昨天问我的一些题，我再看看。

确实能做出来了。

我想我真的进步了。

中午的时候，陈大壮打电话来。

说明天去不去网吧。

我说不去，要复习。

他说你真无趣。

我说没办法。

他说期末考试完了去玩。

我说好。

下午我在家，看《小王子》。

看完了，很喜欢这个故事。

小王子最后回到他的星球，照顾他的玫瑰。

我想我也该照顾好自己的"玫瑰"。

我的玫瑰是什么？

是学习？是梦想？还是林小雅？

我不知道。

傍晚的时候，我站起来活动一下。

窗外面阳光很好。

五月就要结束了。

16岁的第一个月快要过去了。

这个月发生了什么？

生日，总复习开始，和她一起去图书馆。

好像什么都没变，又好像什么都变了。

我们还是朋友，但关系好像更好了。

晚上打开QQ，林小雅的头像亮着。

我发过去："《小王子》看完了。"

她回："感觉怎么样？"

我说："很好看，很喜欢。"

她说："我也很喜欢。"

我说："谢谢你的礼物。"

她说："不用谢。"

然后她说："五月要结束了。"

我说："是啊，时间过得真快。"

她说："六月要期末考试了。"

我说："对，要继续努力。"

她说："一起加油。"

我发了一个加油的表情。

窗外面天黑了。

五月二十八号，周日，五月快结束了。

明天就要六月了。

继续加油吧江小鱼。
